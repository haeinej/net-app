Design system for the March 2026 ohm. spec.

## Principles

- Warm, quiet, analog
- The product recedes so the thought comes forward
- Color communicates depth and warmth only
- No decorative urgency

## Palette

- Warm ground: `#F5F0EA`
- Card ground: `#EDE8E2`
- Vermillion: `#EB4101`
- Olive: `#979C5B`
- Chartreuse: `#D0D37C`
- Panel dark: `#0C0C0A`
- Panel deep: `#080604`
- Type dark: `#1A1A16`
- Type muted: `#9A9A98`
- Type white: `#FFFFFF`

Vermillion has five product uses only:

- full warmth edge bar
- reply label
- notification circle
- post button
- logo period

## Typography

- Display and structural text: Comico in the target spec
- Reading text: Sentient

Runtime note:

- The mobile app now uses Sentient plus Comico directly.

## Layout

- Three screens only
- Full-width 4:3 thought image
- Quiet 2px warmth bar on the left edge
- Inline notification panel inside Worlds
- Bottom nav without labels
