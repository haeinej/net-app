Thought detail and panels.

Every thought has three panels:

- Panel 1: image, sentence, author chip, warmth bar
- Panel 2: sentence anchor plus context
- Panel 3: accepted replies plus reply input

Interaction rules:

- Swipe left to go deeper
- Swipe right to go back
- Each panel gets darker as depth increases
- Reply input is written text only
- Typing should push the warmth bar to full vermillion
- Sending a reply should feel like a deliberate act, not a reaction

Panel 3 constraints:

- Accepted replies only
- Reply placeholder: "what this thought surfaces in you..."
- Replies must pass the acceptance gate before they show up in a conversation
