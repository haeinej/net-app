Phase 7 learning loop.

The system should learn from:

- which replies get accepted
- which conversations sustain
- which cross-domain pairings deepen
- where the user's posting patterns differ from their replying patterns

Hidden health metrics:

- reply-to-post ratio
- conversation depth distribution
- cross-domain connection rate
- new-user first conversation time

The platform should respond to weak health by making resonance easier to find, not by adding urgency mechanics.
