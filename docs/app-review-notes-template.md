## Review overview

ohm. is a mobile app where each person shares one thought at a time. Other people can reply to that thought, and accepted replies open private conversations.

## Reviewer access

- Review account email: `replace-me@example.com`
- Review account password: `replace-me`
- If using a seeded account, include what state the account is already in.

## Core flow to test

1. Sign in with the review account.
2. Complete onboarding if the account is fresh.
3. Create or view a thought.
4. Open replies and accept a reply if needed.
5. Open the private conversation created from an accepted reply.
6. Open `Me > Settings` to review the Support, Privacy Policy, and Delete Account flows.
