Compose flow.

Step order:

- write the sentence
- add up to 600 characters of context
- generate the cinematic image in the background
- post deliberately

Rules:

- The sentence should feel like one thought in motion
- Context should add origin, not closure
- Image should feel like a desaturated film still shaped by the sentence and the person's photo
- Post is not a floating action default; it is a deliberate action

Do not add:

- category chips
- topic tags
- popularity framing
- reaction affordances
