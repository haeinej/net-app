Crossing and Shift.

Availability:

- unlocked after 10 or more exchanges in a conversation
- quiet and discoverable, never prompted

Crossing:

- where two thoughts met
- private shared artifact
- lives on both profiles
- never appears in Worlds
- uses both profile photos to generate the shared image

Shift:

- where thinking moved
- before/after arc for each participant
- can appear in Worlds
- never exposes the private thread itself

Shared rules:

- both people participate
- draft state is silent
- completion is quiet
- abandonment is silent
