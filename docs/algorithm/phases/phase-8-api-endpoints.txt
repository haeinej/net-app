Phase 8 API endpoints.

Core routes:

- `POST /api/thoughts`
- `GET /api/feed`
- `GET /api/thoughts/:id`
- `POST /api/thoughts/:id/reply`
- `GET /api/notifications`
- `POST /api/replies/:id/accept`
- `POST /api/replies/:id/ignore`
- `GET /api/conversations`
- `GET /api/conversations/:id`
- `POST /api/conversations/:id/messages`
- crossing routes
- shift routes
- profile routes

Behavioral rules:

- replies start pending
- accept creates the conversation
- ignore is silent
- delete-thought is soft delete only
- conversations created from a deleted thought remain intact
